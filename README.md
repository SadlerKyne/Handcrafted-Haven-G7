# Handcrafted-Haven-G7
Handcrafted Haven is an innovative web application that provides a platform for artisans and crafters to showcase and sell their unique handcrafted items.

## Purpose
This is a full-stack web app project. "Handcrafted Haven" is the product concept: a marketplace connecting artisans with buyers.

We will be using our technical skills across a full stack (Next.js, Node, database, deployment).

## MileStones

- Seller profiles (authenticated), with story/bio and curated product listings
- Product listings with description, price, images
- Browse/filter by category, price range, other criteria
- Reviews and ratings, open to any user (not just verified buyers, per the doc as written)

## Design/standards:

- Valid, standards-compliant pages: performance, validation, accessibility (WCAG 2.1 AA), SEO, usability
- Responsive design across devices
- Consistent branding (color, typography, layout)
- Clear, logical navigation

## Technical stack:

- Front-end: HTML, CSS, JS, Next.js
- Back-end: Node.js plus a database (MongoDB)
- Git/GitHub for version control, GitHub Boards for project management
- Vercel for deployment
